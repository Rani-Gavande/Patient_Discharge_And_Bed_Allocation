# Patient_Discharge_And_Bed_Allocation
A Hospital Management System designed to streamline patient discharge and bed allocation processes. The system enables hospitals to efficiently manage patient admissions, discharges, and bed availability in real time.

## Project Overview
A hospital management system for patient discharge and bed allocation.
